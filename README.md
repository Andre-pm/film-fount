Um projeto de biblioteca de filmes online construído em Flutter usando o padrão Model-View-ViewModel (MVVM) e os princípios da Arquitetura Limpa.
Você pode acessar a plataforma em: https://film-fount.vercel.app.
<br><br>

### Detalhes técnicos:
**Autenticação e Banco de dados:** Gerenciado via Firebase para lidar com contas de usuário e armazenamento de dados de forma segura.

**Design:** Todas as imagens e logotipos foram criados por mim, e a interface do usuário foi projetada no Figma.

**Fonte de dados:** As informações sobre os filmes são obtidas usando a API TMDB, para mais detalhes, visite https://developer.themoviedb.org/docs/getting-started.

<br>

An online movie library project built in Flutter using the Model-View-ViewModel (MVVM) pattern and Clean Architecture principles.
You can access the platform at: https://film-fount.vercel.app.
<br><br>


### Technical Details:

**Authentication & Database:** Managed via Firebase to handle user accounts and data storage securely.

**Design:** All images and logos were created by me, and the UI was designed in Figma.

**Data Source:** Movie information is fetched using the TMDB API, for more details, visit https://developer.themoviedb.org/docs/getting-started.
